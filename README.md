# SignSense: Real-Time AI Sign Language Translator

## Overview
This is a real-time gesture recognition application developed for our Applied AI Mini Project. It utilizes a trained AI model to translate sign language directly from a webcam feed into text.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **AI Model:** Google Teachable Machine (PoseNet / TensorFlow.js)
* **Deployment:** Vercel

## How to Run Locally
1. Clone this repository.
2. Open the project folder in VS Code.
3. Start a local server (using the Live Server extension) to bypass browser webcam security.
4. Allow camera permissions and start signing!
